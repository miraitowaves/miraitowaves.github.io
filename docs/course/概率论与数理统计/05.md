# 大数定律与中心极限定理

## 切比雪夫不等式
$P\{\vert X - μ \vert \geq \epsilon\} \leq \frac{\sigma^2}{\epsilon^2}$

### 如何理解切比雪夫不等式
- 不管随机变量的分布类型是什么，只要已知它的数学期望和方差，就能够对随机变量落入数学期望附近的区域$(\mu - \epsilon, \mu + \epsilon)$内或外的概率给出一个界的估计
- 所以，计算一个随机变量的数学期望和方差，成为了我们所任何估计的前置要求

## 中心极限定理
$lim_{n\rightarrow + \infty} P\{\frac{\sum_1^nX_i - E(\sum_1^nX_i)}{\sqrt{Var(\sum_1^nX_i)}} \leq x\} = \phi(x)$ 

### 如何理解中心极限定理
- 当n充分大时，$\frac{\sum_1^nX_i - E(\sum_1^nX_i)}{\sqrt{Var(\sum_1^nX_i)}}$近似于标准正太分布
- 中心极限定理仍然需要知道**一个随机变量的数学期望和方差，它同样是在数学期望的附近估计概率**
- 应用：$P\{\sum X_i < 100\} = P\{\frac{\sum_1^nX_i - E(\sum_1^nX_i)}{\sqrt{Var(\sum_1^nX_i)}} < \frac{100 - E(\sum_1^nX_i)}{\sqrt{Var(\sum_1^nX_i)}}\} \approx \phi(\frac{100 - E(\sum_1^nX_i)}{\sqrt{Var(\sum_1^nX_i)}})$
